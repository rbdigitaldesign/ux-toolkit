Jotform

Use case: Surveys, feedback forms, consent forms.

Drag-and-drop interface with templates. Great for student feedback. Auto-generates reports and integrates with Google Sheets and MS Teams.

Optimal Workshop

Use case: Card sorting, tree testing, first-click tests.

Excellent toolset for information architecture validation and navigation usability. Offers clean analytics and session recording.

Visit Optimal Workshop

Hotjar

Use case: Heatmaps, session recordings, user behaviour insights.

Visualise where students click, scroll, and drop off. Useful for diagnosing usability pain points.

Visit Hotjar

Typeform

Use case: Conversational-style surveys and feedback.

Slick, mobile-friendly form layouts ideal for engaging UX-style quizzes or mid-course polls.

Visit Typeform

Google Forms

Use case: Simple, quick surveys with auto-linked results to Google Sheets.

Best for low-friction form delivery in environments already using Google Workspace. Customisable but limited styling.
