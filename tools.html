UX tools for higher education

Curated tools that support user-centred research and testing in digital learning environments.

Accessible Web Helper

Use case: Accessibility audits and WCAG conformance checks.

This Chrome extension scans live webpages for accessibility issues, highlights them visually, and provides WCAG-aligned remediation tips. Great for auditing LMS interfaces or digital course content.

  • Scan in real-time with element highlights
  • Check contrast ratios
  • Streamline audits with RAMP integration
Install on Chrome

Jotform

Use case: Surveys, feedback, UX testing intake forms.

Drag-and-drop builder for capturing student input or consent forms during usability studies. Auto-analyses data and integrates with Google Sheets or MS Teams.

  • Embed forms directly into LMS modules
  • Ideal for pre/post-test and pilot feedback
Visit Jotform

Optimal Workshop

Use case: Card sorting, tree testing, first-click analysis.

Test student navigation behaviours in prototypes or course sites. Use card sorts and tree tests to validate content structure and menu labels.

  • Run open/closed card sorts
  • Analyse success paths and click confidence
Visit Optimal Workshop

Hotjar

Use case: Heatmaps, scrollmaps, session recordings.

See how students interact with pages in real time—where they click, how far they scroll, and what elements they ignore. Ideal for auditing dashboards or homepage usability.

  • Visualise attention zones
  • Flag pain points and dead zones
Visit Hotjar

Typeform

Use case: Conversational-style surveys and student check-ins.

Use Typeform for more engaging, mobile-optimised survey experiences. Useful for gathering deeper feedback at key points in a student journey.

  • Great for micro-polls or sentiment surveys
  • One-question-at-a-time interface
Visit Typeform

Google Forms

Use case: Quick surveys, low-friction feedback.

Fast and familiar, Google Forms is perfect when your institution already uses Google Workspace. It syncs responses into Google Sheets for simple data analysis.

  • Use branching logic for tailored UX tests
  • Embed in LMS or link externally
Visit Google Forms
